    ETIQUETADESCRIPCIÓN
    body Indica la parte del cuerpo del contenido de un documento HTML. Es una etiqueta esencial para cualquier documento ya que indica donde empieza el contenido visible del documento.
    head La parte superior del documento HTML, es donde podremos indicar los metadatos: título del documento, hojas de estilos, javaScript, CSS…
    div Un elemento que es usado mayoritariamente para agrupar otros elementos y actuar como plantilla de otros controles. La etiqueta "div" nos ayuda a estructurar el documento en secciones.
    a Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href, donde pondremos el enlace al que queremos conectar. Otro atributo muy usado es target, el cual nos sirve para indicar si el enlace se abrirá en una nueva ventana o en la misma.
    strong Si tienes mucho texto, es importante poder dar énfasis a una parte en concreto, con la etiqueta strong lo podemos hacer.
    br Con esta etiqueta HTML le podemos decir al navegador que viene un salto de línea. Nos sirve para hacer el texto más leíble.
    H1 hasta H6 Hay diferentes niveles de títulos, del 1 al 6. Las etiquetas < H + número > nos permiten indicar la importancia del título y para estructurar el contenido, de esta forma ayudamos a los bots a entender la importancia del contenido.
    IMG Usamos la etiqueta IMG para mostrar imágenes dentro del contenido. Necesita el atributa src para funcionar, ya que será donde indicaremos desde donde tiene que mostrar la imagen.
    OL - LI Las etiquetas OL y LI nos sirven para crear listas, OL para listas ordenadas y UL para listas sin orden, dentro de las listas, los elementos se identifican con la etiqueta LI.
    P Etiqueta que nos sirve para agrupar texto dentro de un parágrafo. El propósito es poder hacer el contenido más fácil de leer y organizado.
    SPAN Con la etiqueta podemos personalizar el estilo de solamente una parte del texto.

    ESTILOSDESCRIPCIÓN
    Colors Asigna color a un texto o área especifica
    Backgrounds Determina cual va a ser el fondo de un objeto el cual puede ser un color o imagen
    borders Determina el borde de un elemento, color, estilo y grosor
    margins Define la distancia entre 2 elementos desde su orillo externo hacia afuera
    text Acentua las caracteristicas que se van asignar al texto
    fonts Determina el tamaño de la fuente y se puede definir en pixeles o en ems
    Height Señala la altura de un elemento y se puede especificar en pixeles, ems o porcentaje
    width Señala el ancho de un elemento y se puede especificar en pixeles, ems o porcentaje
    Padding Marca la distancia que debe haber desde el borde de un elemento hasta su contenido
    Opacity Hace que un elemento sea transparente
    tables Permite asignar propiedades a una tabla como lineas, colores y fuentes
    position Indica en que posición se va a ubicar un gráfico dentro de un bloque
    Overflow Otorga propiedades de desbordamiento cuando un elemento es demasiado grande
    float Permite otorgar propiedades flotantes a un elemento
